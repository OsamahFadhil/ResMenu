## Moving & Resizing Elements

### **Select Element**
- Click any element to select it
- Blue border appears around it

### **Move Element**
1. Click and hold on element
2. Drag to new position
3. Release mouse

### **Resize Element**
1. Select element (click on it)
2. **8 white circles** appear around it:

3. Drag any circle to resize:

   - **Edges** = resize one side only

### **Rotate Element**
1. Select element
2. See circle with rotation icon at top
3. Drag it to rotate element

### **Delete Element**
1. Select element

## Tips & Tricks ğŸ’¡

### **Alignment:**
- Enable **Grid** (grid lines appear)
- Enable **Snap to Grid** (elements snap to grid)
- Makes everything line up perfectly

### **Duplicating:**
- Select element
- Press **Ctrl+D**
- Duplicate appears slightly offset
- Faster than creating from scratch

### **Color Consistency:**
- Note down color codes you use
- Example: `#dc2626` for red
- Use same colors throughout design
- Creates professional look

### **Layering:**
- Background elements: Send to Back
- Text elements: Bring to Front
- Use layers panel to organize

### **Speed Tips:**
- Use keyboard shortcuts (Ctrl+Z, Ctrl+D, Delete)
- Duplicate elements instead of creating new ones
- Load a template then customize it
- Save frequently (Ctrl+S)

## Troubleshooting

### **Can't select element?**
- Make sure you're using "Select" tool (arrow icon)
- Click directly on the element
- Check if element is locked

### **Can't resize element?**
- Select the element first (click on it)
- Look for 8 white circles around it
- Drag any circle to resize
- Make sure element isn't locked

### **Element disappeared?**
- Check layers panel at bottom
- Element might be behind another element
- Click its name in layers to select
- Use "Bring to Front" button

### **Changes not saving?**
- Click "Save" button after making changes
- Look for "Saved" confirmation

### **Text too small to read?**
- Select text element
- In right panel, increase font size
- Drag resize handles to make text box bigger
